Flower Shower— Instructions

This project shows A generative algorithm the creates simple flower agents that move through a virtual world. The GUI allows users to control their behavior count, size, speed, and direction.

  1. Use the GUI at the top to change the behavior and appearance of the flowers in real time.
  2. Controls:
    • Number of agents: how many flowers appear
    • Agent size: how big each flower is
    • Agent speed: how fast the flowers move
    • Agent direction: what direction the flowers move in
  3. Press X on the keyboard to close these instructions.
